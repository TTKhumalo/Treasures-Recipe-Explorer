# Treasures Recipe Explorer

Welcome to the **Treasures Recipe Explorer**! This is an Angular app that lets users browse, search, and view detailed recipes in an easy-to-navigate interface.

## Features

- Home page with quick links to explore recipes
- Recipe list view (`RecipesComponent`)
- Recipe detail view (`RecipeDetailComponent`)
- Responsive design
- Angular routing for smooth navigation

## Project Structure


src/
└── app/
├── home/ # Home component
├── recipes/ # Recipes list component
├── recipe-detail/ # Recipe detail component

└── app.module.ts
